## 用途

用于识别股票、指数、板块在走势中的关键支撑位和压力位，辅助判断价格回踩、突破、跌破、反弹、止损和止盈位置。

本 Skill 适合用于：

- K 线复盘
- 个股买卖点分析
- 支撑线和压力线绘制
- 趋势突破有效性判断
- 回踩低吸和压力减仓计划
- 结合量价、MA5/MA10、价格行为等指标做综合判断

## 使用边界

- 支撑位和压力位不是固定不变的铁律，而是动态参考区间。
- 支撑位附近不代表一定会涨，压力位附近不代表一定会跌。
- 有效突破或有效跌破，必须结合成交量、收盘价和后续确认。
- 单个点位的意义弱，多次测试、距离更远、市场记忆更强的位置意义更强。
- 支撑压力分析必须结合大盘环境、板块强弱、趋势方向和交易周期。

## 基础定义

支撑位:
  股价下跌时，买盘集中，使下跌暂时受阻的位置。可以理解为“地板”，价格跌到附近容易被托住。

压力位:
  股价上涨时，卖盘集中，使上涨暂时受阻的位置。可以理解为“天花板”，价格涨到附近容易被压住。

支撑线:
  连接两个或多个相近低点形成的水平线或趋势线。

压力线:
  连接两个或多个相近高点形成的水平线或趋势线。

有效突破:
  股价突破压力位，并且收盘站稳压力位上方，最好伴随成交量放大。

有效跌破:
  股价跌破支撑位，并且收盘站在支撑位下方，后续反抽无法收回支撑。

## 通用判断原则

1. 先看位置：支撑和压力必须建立在历史成交密集区、前高、前低、均线、缺口、平台等位置上。
2. 再看次数：测试次数越多、越不容易跌破或突破，该位置越重要。
3. 再看距离：时间跨度越久、距离当前越远但仍有效，市场记忆越强。
4. 再看量能：放量突破压力更可信，放量跌破支撑更危险。
5. 最后看确认：突破后回踩不破，跌破后反抽不过，信号才更稳定。

强弱参考：

- 弱支撑/压力：只出现过一次触碰，且没有明显成交密集。
- 中等支撑/压力：出现两次以上触碰，有一定反应。
- 强支撑/压力：多次测试未破，或者与 MA5/MA10、平台、前高前低、跳空缺口共振。
- 极强支撑/压力：多因素共振，例如前低 + MA10 + 成交密集区。

## 核心规则

### 1. 支撑位

名称:
  支撑位

条件:
  1. 股价下跌到某一区域后多次止跌。
  2. 该区域附近出现买盘承接。
  3. 可以由前低、平台下沿、均线、缺口下沿或成交密集区形成。
  4. 测试次数越多，支撑意义越强。
  5. 跌到支撑附近时，若缩量止跌或出现长下影，支撑更有效。

含义:
  说明该位置有资金愿意买入，价格下跌到附近容易受到托底。支撑位像价格的“地板”，但地板也可能被砸穿。

操作:
  1. 支撑位附近不追空。
  2. 若股价靠近支撑但没有跌破，可观察低吸机会。
  3. 配合缩量回踩、长下影、止跌阳线时，买点更清晰。
  4. 必须设置支撑失效后的风控。

失效信号:
  放量跌破支撑位，并且后续反抽无法站回支撑上方。

### 2. 压力位

名称:
  压力位

条件:
  1. 股价上涨到某一区域后多次遇阻。
  2. 该区域附近出现卖盘抛压。
  3. 可以由前高、平台上沿、均线、缺口上沿或成交密集区形成。
  4. 多次冲击未突破，说明压力较大。
  5. 到压力位时若放量滞涨、长上影，压力更明显。

含义:
  说明该位置有资金愿意卖出，价格上涨到附近容易被压住。压力位像价格的“天花板”，突破前不要默认还能继续涨。

操作:
  1. 压力位附近不盲目追高。
  2. 持仓者可观察是否减仓或止盈。
  3. 若放量突破并站稳，可重新判断为强势。
  4. 若多次冲高失败，要警惕回落。

失效信号:
  股价放量突破压力位，并且回踩压力位不破，压力转为支撑。

### 3. 支撑位买点

名称:
  支撑位买点

条件:
  1. 股价回落到明确支撑位附近。
  2. 股价没有有效跌破支撑。
  3. 支撑附近出现止跌、缩量、长下影或小阳线。
  4. 支撑位来自前低、平台、均线或成交密集区。
  5. 大盘和板块环境没有明显恶化。

含义:
  股价靠近支撑但跌不下去，说明下方承接较强。此时市场风险收益比相对更清晰，可以用支撑位作为防守参考。

操作:
  1. 可在支撑位附近轻仓试错。
  2. 更稳健的做法是等支撑位企稳后再介入。
  3. 买入后以支撑位下方作为止损参考。
  4. 若反弹到压力位附近，要重新评估。

失效信号:
  股价放量跌破支撑位，或连续收盘在支撑位下方。

### 4. 支撑位跌破卖点

名称:
  支撑位跌破卖点

条件:
  1. 股价跌破明确支撑位。
  2. 跌破时成交量放大更危险。
  3. 后续继续跌破新的支撑位。
  4. 反抽时无法重新站回原支撑。
  5. 均线同步走弱，短期趋势转空。

含义:
  原本的买盘托底失效，说明资金承接不足。跌破支撑后，原支撑往往会变成新的压力。

操作:
  1. 持仓者优先控制仓位。
  2. 不在刚跌破时盲目补仓。
  3. 若反抽不过原支撑，可视为二次卖点。
  4. 等新的支撑形成后再考虑修复机会。

失效信号:
  跌破支撑后快速放量收回，并重新站稳支撑位上方。

### 5. 压力位突破买点

名称:
  压力位突破买点

条件:
  1. 股价上涨到明确压力位。
  2. 股价形成有效突破。
  3. 突破时成交量明显放大。
  4. 收盘价站在压力位上方。
  5. 后续回踩压力位不破更可靠。

含义:
  上方卖盘被消化，多头力量占优。原压力位被突破后，往往会变成新的支撑位，行情可能开启新一轮上涨。

操作:
  1. 激进者可在放量突破时轻仓跟随。
  2. 稳健者等突破后回踩不破再买。
  3. 突破后若缩量回踩且不跌回压力位下方，确认更强。
  4. 不追连续加速后的高位假突破。

失效信号:
  突破后快速跌回压力位下方，且成交量放大，说明可能是假突破。

### 6. 压力位遇阻卖点

名称:
  压力位遇阻卖点

条件:
  1. 股价上涨到明确压力位附近。
  2. 多次冲击压力位失败。
  3. 出现长上影、放量滞涨、冲高回落。
  4. 成交量放大但价格无法继续突破。
  5. 短期均线开始走平或拐头向下。

含义:
  上方卖盘较重，多头进攻受阻。如果迟迟不能有效突破，说明资金不愿继续抬高价格，短线可能回落。

操作:
  1. 不追高。
  2. 持仓者可在压力位附近减仓或止盈。
  3. 若后续跌破短期均线，应进一步防守。
  4. 等放量突破压力位后再重新评估。

失效信号:
  股价放量突破压力位，并且收盘站稳压力上方。

### 7. 均线支撑

名称:
  均线支撑

条件:
  1. 股价处于均线之上。
  2. 回踩均线时没有有效跌破。
  3. 均线方向仍然向上。
  4. ETF 分析 skill 内只参考 MA5 和 MA10。
  5. MA5/MA10 与前低、平台下沿或成交密集区共振时更有效。

含义:
  均线代表一段时间内的市场成本。股价回踩均线不破，说明平均成本附近有资金承接，趋势仍偏强。

操作:
  1. 上升趋势中，回踩 MA5 或 MA10 不破可关注承接。
  2. MA10 向上且回踩不破，支撑有效性更强。
  3. 均线支撑要结合成交量，不要只看线。
  4. 若股价远离均线太多，不适合追高。

失效信号:
  股价放量跌破均线，且均线开始走平或拐头向下。

### 8. 均线压力

名称:
  均线压力

条件:
  1. 股价处于均线之下。
  2. 反弹到均线附近遇阻。
  3. 均线方向向下。
  4. 反弹成交量不足，无法站上均线。
  5. 跌破均线后多次反抽不过，压力更强。

含义:
  均线从支撑变成压力，说明市场平均成本压在股价上方。反弹到均线附近容易遇到解套盘和卖盘。

操作:
  1. 下降趋势中，不把反弹到均线当成强势突破。
  2. 反弹到下行均线遇阻，可考虑减仓。
  3. 等放量站上均线并回踩不破后，再判断转强。
  4. 不在均线压力下方重仓抄底。

失效信号:
  股价放量站上均线，并且均线逐步走平或拐头向上。

### 9. 前高压力

名称:
  前高压力

条件:
  1. 历史走势中存在明显阶段高点。
  2. 股价上涨到前高附近。
  3. 前高区域多次未能突破，压力更大。
  4. 若前高位置成交密集，压力更强。
  5. 若突破前高时成交量放大，突破更可信。

含义:
  前高是天然的心理关口。很多资金会在前高附近选择解套、止盈或观察，所以价格容易遇阻。

操作:
  1. 前高附近不盲目追涨。
  2. 多次冲击前高不过，要警惕回落。
  3. 放量突破前高后，形态会明显转强。
  4. 突破后回踩前高不破，可视为确认信号。

失效信号:
  放量突破前高后又快速跌回前高下方，说明突破失败。

### 10. 前低支撑

名称:
  前低支撑

条件:
  1. 历史走势中存在明显阶段低点。
  2. 股价回落到前低附近。
  3. 多次探底回升，支撑更强。
  4. 时间间隔越久、测试次数越多，支撑越有参考价值。
  5. 若前低附近出现缩量止跌，支撑更可靠。

含义:
  前低是天然的心理关口。此前价格在这里被资金托住，后续再次回到附近时，市场会观察这里是否还能承接。

操作:
  1. 前低附近可观察止跌信号。
  2. 不在没有企稳前直接重仓买入。
  3. 若多次探底回升，可视为强支撑。
  4. 跌破前低后，要按支撑失效处理。

失效信号:
  股价放量跌破前低，并持续在前低下方运行。

### 11. 向上跳空缺口支撑

名称:
  向上跳空缺口支撑

条件:
  1. 股价高开形成向上跳空缺口。
  2. 缺口下沿没有被快速回补。
  3. 缺口出现后股价继续走强。
  4. 3 到 5 个交易日内缺口不回补，支撑意义增强。
  5. 缺口附近伴随放量，说明资金情绪较强。

含义:
  向上跳空缺口是资金急着抢筹形成的区域。缺口下沿常被视为支撑，因为资金不愿等更低价格。

操作:
  1. 缺口不回补时，可把缺口下沿作为支撑参考。
  2. 回踩缺口不破，可关注低吸机会。
  3. 若缺口被完全回补，要降低强势预期。
  4. 缺口越大，越要注意短线情绪过热。

失效信号:
  3 到 5 个交易日内快速回补缺口，并跌破缺口下沿。

### 12. 向下跳空缺口压力

名称:
  向下跳空缺口压力

条件:
  1. 股价低开形成向下跳空缺口。
  2. 缺口上沿没有被快速回补。
  3. 缺口出现后股价继续走弱。
  4. 3 到 5 个交易日内缺口不回补，压力意义增强。
  5. 缺口附近反弹无力，说明上方抛压重。

含义:
  向下跳空缺口是资金恐慌卖出形成的区域。缺口上沿常被视为压力，因为资金不愿等更高价格。

操作:
  1. 反弹到缺口附近不追高。
  2. 缺口上沿遇阻，可考虑减仓。
  3. 放量回补缺口后，才说明压力减轻。
  4. 缺口未补前，短线反弹按弱修复看待。

失效信号:
  股价放量回补缺口，并站上缺口上沿。

### 13. 支撑压力互换

名称:
  支撑压力互换

条件:
  1. 原压力位被有效突破。
  2. 股价回踩原压力位不破。
  3. 原压力位转化为新支撑位。
  4. 或原支撑位被有效跌破。
  5. 股价反抽原支撑位不过，原支撑位转化为新压力位。

含义:
  市场对同一价格区域的预期发生变化。突破后，原来想卖的人减少，新的买盘开始承接；跌破后，原来想买的人失去信心，反抽时容易卖出。

操作:
  1. 压力突破后，等回踩不破再确认新支撑。
  2. 支撑跌破后，等反抽不过再确认新压力。
  3. 支撑压力互换比单次突破更可靠。
  4. 结合成交量判断真假突破。

失效信号:
  突破后快速跌回压力位下方，或跌破后快速收回支撑位上方。

## 画线方法

1. 切换到适合交易周期的 K 线图，例如日线、60 分钟线、周线。
2. 使用证券 APP 自带的画线工具。
3. 画支撑线时，至少连接两个有效低点。
4. 画压力线时，至少连接两个有效高点。
5. 水平线适合平台、前高、前低；趋势线适合上升趋势或下降趋势。
6. 随着股价波动动态调整，原有线失效后及时重画。
7. 放量突破或放量跌破时，原支撑压力要重新评估。

画线注意：

- 支撑和压力更适合看“区域”，不要只盯一个精确价格。
- 影线可以作为参考，但收盘价更重要。
- 多周期共振更可靠，例如日线支撑和周线支撑接近。
- 支撑压力结合 MA5/MA10、量能、K 线形态和价格行为更有参考价值。

## 综合使用流程

1. 找位置：先标出前高、前低、平台上沿、平台下沿、跳空缺口和 MA5/MA10。
2. 画区域：把支撑压力看成一个价格区间，不要只看一条线。
3. 看反应：观察价格到达支撑或压力后的 K 线和量能变化。
4. 看突破：突破压力要看是否放量，跌破支撑也要看是否放量。
5. 看确认：突破后回踩不破，跌破后反抽不过，信号更有效。
6. 定计划：支撑附近考虑买入，压力附近考虑卖出，突破压力可能开启新上涨，跌破支撑可能开启新下跌。
7. 做风控：每个判断都要提前设定失效信号。

## 风险提示

支撑位和压力位是技术分析中的辅助工具，不是确定性预测。市场随时变化，任何支撑都可能跌破，任何压力都可能突破。实际使用时，应结合成交量、MA5/MA10、K 线形态、价格行为、板块强弱、大盘环境和仓位管理综合判断。
