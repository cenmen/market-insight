## 用途

用于识别股票、指数、板块、期货、外汇等价格走势中的趋势方向、趋势支撑、趋势压力、突破信号和转势风险。

本 Skill 适合用于：

- K 线复盘
- 趋势方向判断
- 趋势线画法规范
- 趋势支撑和趋势压力识别
- 趋势突破与跌破判断
- 趋势交易中的买点、卖点、止损位规划
- 结合量价、均线、支撑压力位做综合分析

## 使用边界

- 趋势线是辅助工具，不是绝对预测。
- 画趋势线要尽量客观，不能为了符合观点强行画线。
- 趋势线更适合看“区域”，不要只盯一个精确价格。
- 趋势线被突破或跌破后，需要结合收盘价、成交量和后续确认。
- 级别越大的趋势线越重要，周线趋势线通常比日线趋势线更强，日线趋势线通常比分时趋势线更强。
- 趋势线必须跟随市场变化动态调整，旧趋势失效后要重新画线。

## 基础定义

趋势:
  价格在一段时间内持续朝某个方向运行的状态。常见趋势包括上涨趋势、下跌趋势和横盘趋势。

上升趋势:
  股价低点不断抬高，高点也逐步抬高，说明多头占优。

下降趋势:
  股价高点不断降低，低点也逐步降低，说明空头占优。

横盘趋势:
  股价在一个区间内来回震荡，上方有压力，下方有支撑，多空暂时平衡。

上升趋势线:
  连接两个或多个依次抬高的低点形成的线，主要起支撑作用。

下降趋势线:
  连接两个或多个依次降低的高点形成的线，主要起压力作用。

趋势通道:
  在趋势线基础上，画出一条平行线，形成价格运行的通道区间。

## 通用判断原则

1. 先看趋势方向：上涨看低点是否抬高，下跌看高点是否降低。
2. 再找关键点：上升趋势线连接低点，下降趋势线连接高点。
3. 再看触碰次数：两个点只能画出候选趋势线，三次以上触碰更有参考价值。
4. 再看量能配合：放量突破或放量跌破更有效，缩量突破容易失败。
5. 最后看后续确认：突破后回踩不破，跌破后反抽不过，趋势变化更可信。

强弱参考：

- 弱趋势线：只有两个连接点，时间跨度短。
- 中等趋势线：至少三个触碰点，价格多次沿趋势线运行。
- 强趋势线：触碰次数多、时间跨度长、与均线或支撑压力共振。
- 极强趋势线：大周期有效，并且多次回踩或反抽都没有破坏。

## 标准输出格式

```md
名称:
  规则名称

条件:
  1. 条件一
  2. 条件二
  3. 条件三

含义:
  对当前趋势方向、多空力量和价格状态的解释

操作:
  1. 可以采取的动作
  2. 需要等待的确认
  3. 需要规避的风险

失效信号:
  规则判断不成立或需要改变预期的情况
```

## 核心规则

### 1. 上升趋势线

名称:
  上升趋势线

条件:
  1. 股价处于上涨趋势中。
  2. 至少出现两个依次抬高的低点。
  3. 用直线连接这些低点，并向右延伸。
  4. 后续股价回踩该线附近能够止跌。
  5. 第三次触碰不破，趋势线有效性增强。

含义:
  上升趋势线代表多头的防守线。价格每次回落到趋势线附近都有资金承接，说明上涨结构仍然存在。

操作:
  1. 趋势线附近不追空。
  2. 回踩趋势线不破，可观察低吸机会。
  3. 配合缩量回踩、长下影、止跌阳线，买点更清晰。
  4. 跌破趋势线后，要降低上涨趋势预期。

失效信号:
  股价放量跌破上升趋势线，并且后续反抽无法重新站回趋势线上方。

### 2. 下降趋势线

名称:
  下降趋势线

条件:
  1. 股价处于下跌趋势中。
  2. 至少出现两个依次降低的高点。
  3. 用直线连接这些高点，并向右延伸。
  4. 后续股价反弹到该线附近遇阻。
  5. 第三次触碰不过，趋势线有效性增强。

含义:
  下降趋势线代表空头的压制线。价格每次反弹到趋势线附近都有卖盘压制，说明下跌结构仍然存在。

操作:
  1. 趋势线下方不盲目重仓抄底。
  2. 反弹到下降趋势线附近遇阻，可考虑减仓。
  3. 若放量突破下降趋势线，可重新评估趋势修复。
  4. 突破前，把反弹先当作弱修复看待。

失效信号:
  股价放量突破下降趋势线，并且回踩趋势线不破。

### 3. 横盘趋势线

名称:
  横盘趋势线

条件:
  1. 股价在某一区间内反复震荡。
  2. 上方高点相对接近，形成区间上沿。
  3. 下方低点相对接近，形成区间下沿。
  4. 区间内至少出现两次上沿遇阻和两次下沿支撑。
  5. 成交量没有明显持续放大，市场处于等待方向阶段。

含义:
  多空在区间内反复拉扯，趋势方向暂时不清晰。上沿是压力，下沿是支撑，真正方向要等突破或跌破。

操作:
  1. 区间下沿附近看支撑反应。
  2. 区间上沿附近看压力反应。
  3. 放量突破上沿偏多。
  4. 放量跌破下沿偏空。

失效信号:
  股价有效突破区间上沿或有效跌破区间下沿，横盘结构结束。

### 4. 趋势线画法

名称:
  趋势线画法

条件:
  1. 先确定分析周期，如日线、周线、60 分钟线。
  2. 上升趋势线连接依次抬高的低点。
  3. 下降趋势线连接依次降低的高点。
  4. 至少连接两个关键点，第三个触碰点用于确认。
  5. 尽量连接市场真正反应明显的位置，不强行贴合所有影线。

含义:
  趋势线的核心是找到市场反复认可的支撑或压力区域。画线越客观，后续判断越有参考意义。

操作:
  1. 优先连接明显拐点。
  2. 影线和实体都可参考，但收盘价更重要。
  3. 不要为了让走势看起来好看而随意调整线。
  4. 趋势改变后及时重画。

失效信号:
  原趋势线被价格多次穿越，已经无法对价格形成支撑或压力。

### 5. 有效趋势线确认

名称:
  有效趋势线确认

条件:
  1. 趋势线至少由两个关键点画出。
  2. 后续出现第三次触碰并产生明显反应。
  3. 触碰趋势线时，价格没有有效突破或跌破。
  4. 趋势线持续时间越长，有效性越高。
  5. 趋势线与均线、前高前低、成交密集区共振时更可靠。

含义:
  趋势线被市场多次验证，说明该线附近存在稳定的买卖力量。触碰次数越多，市场参与者越容易形成共同预期。

操作:
  1. 第三次触碰不破后，可提高趋势线权重。
  2. 结合量能判断支撑或压力强度。
  3. 多周期共振时，优先尊重大周期趋势线。
  4. 不把第一次画出的线当成强信号。

失效信号:
  趋势线被放量突破或放量跌破，并且后续确认失败。

### 6. 上升趋势线支撑买点

名称:
  上升趋势线支撑买点

条件:
  1. 股价处于上升趋势中。
  2. 回调靠近上升趋势线。
  3. 股价没有有效跌破趋势线。
  4. 回踩时成交量缩小，说明抛压减轻。
  5. 趋势线附近出现止跌 K 线，如长下影、小阳线、看涨吞没。

含义:
  上涨趋势中的正常回踩，趋势线附近有资金承接。只要趋势线不破，原上涨结构仍然偏健康。

操作:
  1. 可在趋势线附近轻仓试错。
  2. 更稳健的做法是等止跌 K 线确认后介入。
  3. 以趋势线下方作为风控参考。
  4. 若反弹到通道上沿或前高，重新评估止盈。

失效信号:
  回踩后放量跌破上升趋势线。

### 7. 下降趋势线压力卖点

名称:
  下降趋势线压力卖点

条件:
  1. 股价处于下降趋势中。
  2. 反弹靠近下降趋势线。
  3. 股价无法有效突破趋势线。
  4. 反弹过程中成交量不足，或放量滞涨。
  5. 趋势线附近出现长上影、看跌吞没、冲高回落。

含义:
  下跌趋势中的反弹遇阻，说明上方卖盘仍然较强。趋势线没有突破前，反弹更像弱修复。

操作:
  1. 不在下降趋势线下方追高。
  2. 持仓者可在趋势线附近减仓。
  3. 等放量突破并回踩不破后，再判断趋势转强。
  4. 若遇阻后跌破短期均线，应进一步防守。

失效信号:
  股价放量突破下降趋势线，并且收盘站稳趋势线上方。

### 8. 跌破上升趋势线

名称:
  跌破上升趋势线

条件:
  1. 股价原本沿上升趋势线运行。
  2. 某日跌破上升趋势线。
  3. 跌破时成交量放大更危险。
  4. 收盘价站在趋势线下方。
  5. 后续反抽无法重新站回趋势线。

含义:
  原上涨结构被破坏，多头防守失败。趋势可能从上涨转为震荡，甚至进入下跌。

操作:
  1. 持仓者优先控制仓位。
  2. 不在刚跌破时盲目补仓。
  3. 若反抽不过原趋势线，可视为二次卖点。
  4. 等新支撑或新趋势形成后再考虑。

失效信号:
  跌破后快速放量收回，并重新站上上升趋势线。

### 9. 突破下降趋势线

名称:
  突破下降趋势线

条件:
  1. 股价原本受下降趋势线压制。
  2. 某日突破下降趋势线。
  3. 突破时成交量明显放大。
  4. 收盘价站在趋势线上方。
  5. 后续回踩趋势线不破。

含义:
  原下跌结构被打破，空头压制减弱。若突破后能够站稳，可能进入反弹或新一轮上涨趋势。

操作:
  1. 激进者可在放量突破时轻仓试错。
  2. 稳健者等回踩不破后再介入。
  3. 观察突破后能否形成更高的低点。
  4. 若突破后缩量回落，要防假突破。

失效信号:
  突破后很快跌回下降趋势线下方。

### 10. 趋势线反抽确认

名称:
  趋势线反抽确认

条件:
  1. 原趋势线被突破或跌破。
  2. 股价随后回到趋势线附近测试。
  3. 突破后的回踩不破，原压力转支撑。
  4. 跌破后的反抽不过，原支撑转压力。
  5. 确认时成交量配合更可靠。

含义:
  趋势线角色发生变化。原来的压力被突破后可能变成新支撑，原来的支撑被跌破后可能变成新压力。

操作:
  1. 突破后等回踩不破，可作为确认买点。
  2. 跌破后等反抽不过，可作为确认卖点。
  3. 反抽确认比单日突破更稳。
  4. 若确认失败，及时修正判断。

失效信号:
  突破后回踩跌回趋势线下方，或跌破后反抽重新站上趋势线。

### 11. 假突破与假跌破

名称:
  假突破与假跌破

条件:
  1. 股价短暂突破或跌破趋势线。
  2. 突破或跌破时成交量不足。
  3. 收盘价没有有效站稳趋势线外侧。
  4. 后续 1 到 3 个交易日快速回到原趋势内。
  5. 常见于震荡行情、洗盘、诱多或诱空阶段。

含义:
  市场短暂制造方向信号，但没有形成真正趋势改变。假突破容易诱多，假跌破容易诱空。

操作:
  1. 不只看盘中突破，要看收盘确认。
  2. 突破没量时保持谨慎。
  3. 观察 1 到 3 个交易日是否站稳。
  4. 若快速回到原趋势内，要及时修正方向。

失效信号:
  股价突破或跌破后持续站稳，并且成交量逐步放大。

### 12. 趋势通道

名称:
  趋势通道

条件:
  1. 先画出主趋势线。
  2. 再画一条与主趋势线平行的线。
  3. 上升通道中，下轨是支撑，上轨是压力。
  4. 下降通道中，上轨是压力，下轨是支撑。
  5. 股价多次在通道内运行，通道有效性增强。

含义:
  趋势通道描述价格在趋势中的运行范围。通道下沿适合观察承接，通道上沿适合观察抛压。

操作:
  1. 上升通道下沿附近看低吸机会。
  2. 上升通道上沿附近看止盈压力。
  3. 下降通道上沿附近看减仓压力。
  4. 放量突破通道上沿或跌破通道下沿，需要重新评估趋势级别。

失效信号:
  股价连续脱离通道运行，原通道无法约束价格。

### 13. 趋势线角度

名称:
  趋势线角度

条件:
  1. 趋势线斜率反映上涨或下跌速度。
  2. 角度太平，说明趋势力量较弱。
  3. 角度适中，说明趋势更稳定。
  4. 角度太陡，说明短期加速明显。
  5. 加速后若跌破陡峭趋势线，容易出现快速回调。

含义:
  趋势线角度代表价格运行节奏。太陡的趋势线往往难以长期维持，太平的趋势线则说明趋势效率不高。

操作:
  1. 稳健趋势优先选择角度适中的趋势线。
  2. 对陡峭上涨趋势保持追高警惕。
  3. 加速趋势跌破短期趋势线时，先防回调。
  4. 趋势加速后可重新画更贴近价格的新趋势线。

失效信号:
  趋势节奏明显变化，原角度无法描述当前走势。

### 14. 多周期趋势线

名称:
  多周期趋势线

条件:
  1. 同时观察周线、日线、60 分钟线等不同周期。
  2. 大周期趋势决定主要方向。
  3. 小周期趋势用于寻找具体买卖点。
  4. 多周期趋势线方向一致时，信号更强。
  5. 小周期突破若与大周期相反，持续性通常较弱。

含义:
  多周期共振可以减少单周期误判。大周期看方向，小周期找节奏，是趋势线实战中更稳的用法。

操作:
  1. 先看周线或日线确定大方向。
  2. 再看 60 分钟线或 30 分钟线找入场位置。
  3. 大周期上升，小周期回踩支撑更适合关注。
  4. 大周期下降，小周期反弹压力更适合防守。

失效信号:
  大周期趋势线被有效突破或跌破，原多周期判断需要重做。

## 综合使用流程

1. 选周期：先确定用日线、周线还是分钟线分析。
2. 判趋势：上涨看低点抬高，下跌看高点降低，横盘看区间震荡。
3. 找拐点：上升趋势连接低点，下降趋势连接高点。
4. 画趋势线：至少连接两个点，第三个点用于确认。
5. 看量能：突破或跌破时是否放量。
6. 看确认：突破后回踩不破，跌破后反抽不过。
7. 定策略：上升趋势线附近看支撑，下降趋势线附近看压力。
8. 做风控：每一条趋势线都要有失效条件。
9. 动态调整：趋势变化后及时重画，不死守旧线。

## 常见误区

1. 为了证明自己的观点，强行画出一条看似有效的线。
2. 只连接影线极端点，忽略收盘价和真实成交密集区。
3. 把两个点画出的候选趋势线当成强趋势线。
4. 只看盘中突破，不看收盘是否站稳。
5. 突破不看成交量，容易把假突破当真突破。
6. 趋势线失效后仍然不调整。
7. 小周期趋势线和大周期趋势线冲突时，只选择对自己有利的周期。

## 风险提示

趋势线是技术分析中的辅助工具，不是确定性买卖信号。实际使用时，应结合成交量、均线、支撑压力位、K 线形态、MACD、板块强弱、大盘环境和仓位管理综合判断。任何趋势线都可能被突破或跌破，关键是提前设定失效条件。
