## 用途

用于识别股票、指数、板块在不同阶段的量价关系，辅助判断当前走势是资金承接、趋势延续、洗盘、出货，还是下跌加速。

本 Skill 适合用于：

- K 线复盘
- 个股走势解读
- 板块强弱判断
- 短线交易计划制定
- 识别追高、抄底、洗盘、派发风险

## 使用边界

- 量价关系只能辅助判断，不能单独作为买卖依据。
- 必须结合位置、趋势、均线、题材强度、大盘环境一起看。
- 同一种量价形态，在低位、中位、高位的含义可能完全不同。
- 放量不一定是好事，缩量也不一定是坏事，关键看价格所处位置和后续确认。

## 通用判断原则

1. 先看位置：低位、中位、高位，决定形态含义。
2. 再看趋势：上升趋势、震荡趋势、下降趋势，决定操作优先级。
3. 再看量能：与 5 日、10 日、20 日均量对比，判断资金活跃度。
4. 最后看结果：放量后能不能涨，缩量后能不能稳，突破后能不能站住。

常用量能参考：

- 温和放量：当日成交量大于 5 日均量 1.2 倍。
- 明显放量：当日成交量大于 5 日均量 1.5 倍。
- 巨量：当日成交量大于 5 日均量 2 倍，或创近 20 日以上新高。
- 缩量：当日成交量低于 5 日均量 80%。
- 地量：当日成交量接近近 20 日最低水平。

## 标准输出格式

```md
名称:
  形态名称

条件:
  1. 条件一
  2. 条件二
  3. 条件三

含义:
  对当前资金行为和走势状态的解释

操作:
  1. 可以采取的动作
  2. 需要等待的确认
  3. 需要规避的风险

失效信号:
  形态判断不成立或需要改变预期的情况
```

## 九种量价关系

### 1. 放量下跌

名称:
  放量下跌

条件:
  1. 当日成交量明显放大，通常大于 5 日均量 1.5 倍。
  2. 股价当日下跌，或盘中出现明显杀跌。
  3. 常见于股价高位，或已经处在下降趋势中。
  4. K 线实体较大，或盘中抛压持续增强。
  5. 若跌破关键支撑位，风险信号更强。

含义:
  短期抛压明显变大，资金分歧加剧。如果出现在高位，可能是主力出货或筹码松动；如果出现在下降趋势中，往往说明下跌开始加速。

操作:
  1. 不急着抄底。
  2. 持仓者优先控制仓位。
  3. 等放量下跌后能否缩量企稳，再判断是否有修复机会。
  4. 若连续放量下跌，先按风险释放处理。

失效信号:
  放量下跌后很快收回关键支撑，并出现放量反包，说明可能是假跌破或洗盘。

### 2. 缩量新低

名称:
  缩量新低

条件:
  1. 股价沿下降趋势持续走低。
  2. 股价创出阶段新低。
  3. 成交量逐步缩小，接近近 20 日低量区。
  4. 下跌速度开始放缓。
  5. 市场关注度下降，买卖都不活跃。

含义:
  交易开始清淡，抛压比前期减轻。出现新低但没有明显放量，说明恐慌盘可能减少。也可能是主力故意打压股价，等待低位吸筹。

操作:
  1. 不因为新低就马上割肉，也不因为缩量就立刻抄底。
  2. 观察后续是否出现止跌 K 线。
  3. 等放量回升、站回短期均线后再确认。
  4. 适合加入观察池，不适合盲目重仓。

失效信号:
  缩量新低后继续放量破位，说明抛压重新放大，下跌可能还没结束。

### 3. 增量回升

名称:
  增量回升

条件:
  1. 股价前期经历下跌或充分调整。
  2. 成交量开始逐步放大。
  3. 股价从低位企稳反弹。
  4. 突破重要压力位，如前高、平台上沿、短期均线。
  5. 放量上涨后没有快速回落。

含义:
  买盘力量开始增强，资金愿意进场承接。若能突破关键压力位，说明前期洗盘或建仓可能接近尾声，行情有进入反弹或新一轮上升的可能。

操作:
  1. 关注突破有效性。
  2. 激进者可在放量突破时轻仓试错。
  3. 稳健者等回踩不破压力转支撑后再考虑。
  4. 若反弹没有持续放量，不宜追太高。

失效信号:
  放量回升后次日快速缩量回落，并跌回突破位下方，说明突破可能失败。

### 4. 回头确认

名称:
  回头确认

条件:
  1. 前期已经出现放量上涨。
  2. 随后股价缩量回调。
  3. 回调没有跌破重要支撑位。
  4. 回调没有跌破前期低点。
  5. 股价在支撑附近开始企稳。

含义:
  这通常是上涨后的洗盘确认。缩量说明抛压不强，支撑没有跌破说明承接还在。如果确认成功，后续可能继续沿原趋势上行。

操作:
  1. 重点看支撑位能不能守住。
  2. 回踩缩量、企稳后可关注低吸机会。
  3. 若再次放量上攻，可视为确认完成。
  4. 不在刚开始回调时急着买，等企稳更稳妥。

失效信号:
  回调过程中突然放量跌破支撑，说明洗盘可能变成出货或趋势转弱。

### 5. 新量新价有新高

名称:
  新量新价有新高

条件:
  1. 成交量创阶段新高，或明显高于近期均量。
  2. 股价同步创出阶段新高。
  3. 股价突破平台、前高或重要压力位。
  4. 当日收盘尽量站在突破位上方。
  5. 题材、板块或大盘环境配合更好。

含义:
  这是典型的量价齐升形态，说明市场交易情绪较高，资金认可当前方向。如果位置不算太高，可能进入加速上涨阶段。

操作:
  1. 重点观察突破后的承接。
  2. 不宜在连续大涨后无脑追高。
  3. 若次日高开不回落，说明强势延续。
  4. 若回踩突破位不破，可作为二次确认点。

失效信号:
  新高后放量滞涨、冲高回落，或次日直接低开跌回突破位下方，说明可能是假突破。

### 6. 缩量回头

名称:
  缩量回头

条件:
  1. 股价处于上升趋势中。
  2. 回调时成交量逐步缩小。
  3. 股价靠近短期均线或趋势支撑后止跌。
  4. 回调幅度不深，没有破坏上升结构。
  5. 随后出现企稳或小阳线修复。

含义:
  主力没有明显出货，市场抛压不强。缩量回调更多像上涨中继，说明资金还在，趋势可能继续。

操作:
  1. 关注短期均线附近的承接。
  2. 回调缩量不破趋势线，可考虑低吸。
  3. 再次放量上攻时确认趋势延续。
  4. 若已经高位连续加速，仍需降低预期。

失效信号:
  缩量回调后突然放量跌破均线或趋势线，说明原来的上升结构被破坏。

### 7. 一根巨量

名称:
  一根巨量

条件:
  1. 当日成交量突然大幅放大。
  2. 成交量明显高于前期，通常大于 5 日均量 2 倍。
  3. 常见于高位、突破位、消息刺激日或情绪高潮日。
  4. 股价可能大涨、滞涨，也可能冲高回落。
  5. 若巨量后价格没有继续上涨，风险更高。

含义:
  短期资金分歧非常大。尤其在高位出现巨量，同时股价不涨、滞涨或回落，往往说明有资金趁热出货。

操作:
  1. 不单纯因为巨量就判断利好。
  2. 先看巨量当天的收盘位置。
  3. 再看次日能否继续放量上攻。
  4. 若巨量后缩量回落，优先防守。

失效信号:
  巨量后继续放量上涨，并且股价站稳新高，说明巨量可能是换手而不是出货。

### 8. 有价无量

名称:
  有价无量

条件:
  1. 股价继续上涨或创出新高。
  2. 成交量没有同步放大，反而逐步缩小。
  3. 上涨动能开始减弱。
  4. 常见于上涨后期或高位震荡区。
  5. 若配合长上影、小实体 K 线，背离更明显。

含义:
  股价还在涨，但量能跟不上，属于量价背离。说明追涨资金开始不足。如果出现在高位，可能是主力拉高诱多，也可能是上涨趋势进入衰竭阶段。

操作:
  1. 不追高。
  2. 持仓者观察是否跌破短期均线。
  3. 若后续补量上涨，可以继续看趋势。
  4. 若缩量上涨后放量下跌，要及时降低仓位。

失效信号:
  缩量上涨后突然放量突破，并且收盘站稳新高，说明资金重新进场，背离暂时解除。

### 9. 顶部放量

名称:
  顶部放量

条件:
  1. 近 20 日涨幅大于 20%。
  2. 股价处于阶段高位，并创出阶段新高。
  3. 上涨已经经历加速阶段。
  4. 当日成交量大于 5 日均量 2 倍。
  5. 当日收盘涨幅小于 3%，或出现放量滞涨。
  6. 出现长上影、冲高回落、高位震荡等信号。

含义:
  高位放量但价格涨不动，说明筹码分歧很大。很可能是主力在高位派发筹码，也可能是短线资金集中兑现。

操作:
  1. 不追高。
  2. 持仓者优先考虑减仓或锁定利润。
  3. 等回踩确认，不在高位巨量当天冲进去。
  4. 若后续跌破 5 日线或平台支撑，要按顶部风险处理。

失效信号:
  顶部放量后继续强势上涨，并且连续站稳新高，说明高位换手成功，原顶部判断需要修正。

## 综合使用流程

1. 判断位置：先分清低位、半山腰、高位。
2. 判断趋势：看股价是在上升、震荡，还是下降。
3. 判断量能：与 5 日、10 日、20 日均量比较。
4. 匹配形态：从九种量价关系中选择最接近的情况。
5. 等待确认：不要只看单日 K 线，至少看后续 1 到 3 个交易日。
6. 制定操作：上涨趋势重确认，下降趋势重风控，高位巨量重警惕。

## 风险提示

量价关系是经验规则，不是确定性信号。实盘中还要结合大盘环境、板块强弱、消息面、流动性、个股基本面和交易纪律。任何形态都可能失败，关键是提前想好失效条件和仓位控制。
